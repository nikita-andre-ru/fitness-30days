## 🎯 Концепция приложения

Система, где каждый день добавляется **новое упражнение на 30 дней**:

```
День 1:  1 упражнение (5 минут)
День 2:  2 упражнения (10 минут)  
День 3:  3 упражнения (15 минут)
...
День 30: 30 упражнений (50 минут)
```

**Главное:** Данные синхронизируются на **все устройства** автоматически! ☁️

## 🔄 Синхронизация данных

### Как это работает:

```
Компьютер (Chrome)
    ↓ добавляешь упражнение
    ↓ сохраняется в Firebase
    ↓
☁️ Firebase облако
    ↑ читает данные
    ↑
Телефон (Safari)
    ↑ видишь то же упражнение!
```

### Места хранения:

```
Firebase Database: /users/{uid}/exercises/{id}
├── name (название упражнения)
├── link (ссылка на YouTube)
├── startDate (дата добавления)
├── completed (дни выполнения)
└── reps (количество повторений)
```

## 📞 Поддержка

### Если что-то не работает:
1. Проверь что все 4 файла загружены на GitHub
2. Убедись что Vercel развернул приложение
3. Перезагрузи страницу приложения (Ctrl+Shift+R)
4. Проверь консоль браузера (F12 → Console)

### Частые проблемы:

**"Ошибка входа"**
- Убедись что включена аутентификация Google в Firebase
- Проверь что используется правильная почта

**"Данные не синхронизируются"**
- Убедись что Realtime Database создана
- Проверь правила безопасности

**"Черный экран"**
- Проверь консоль браузера (F12)
- Убедись что конфигурация Firebase правильная
