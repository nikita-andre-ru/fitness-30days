```
Папка проекта:
```

---

## 🎯 ШАГ 1: Скачай все файлы

Скачай эти 4 файла:
1. `fitness_app_firebase.html`
2. `package.json`
3. `server.js`
4. `vercel.json`

И положи их в одну папку (назови папку `fitness-30days`)

---

## 🎯 ШАГ 2: Создай репозиторий на GitHub

1. Открой: https://github.com/new
2. **Repository name**: `fitness-30days`
3. Нажми **"Create repository"**
4. Следуй инструкциям GitHub:
   ```bash
   # Вместо /path/to/fitness-30days напиши твой путь
   cd /path/to/fitness-30days
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/fitness-30days.git
   git push -u origin main
   ```

---

## 🎯 ШАГ 3: Развернуть на Vercel

1. Открой: https://vercel.com/new
2. Нажми **"Import Project"**
3. Вставь ссылку на GitHub репозиторий:
   ```
   https://github.com/YOUR_USERNAME/fitness-30days
   ```
4. Нажми **"Import"**
5. Vercel автоматически создаст приложение
6. Жди 30 секунд...
7. **Готово! ✅**

Vercel даст тебе ссылку типа:
```
https://fitness-30days.vercel.app
```

## 💡 Если не хочешь GitHub:

Можешь просто загрузить на Vercel без GitHub:

4. Vercel развернет автоматически
